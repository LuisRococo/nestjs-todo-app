# TODO APP

TODO APP using the following stack:

- ReactJS with TS
- Jest
- Storybook
- Material UI
- NestJS
- SQL Database

# SETUP

1. Create `.env`

```
$ cp .env.template .env
```

2. Run docker

```
$ docker compose up
```
