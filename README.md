# TODO APP

TODO APP using the following stack:

- ReactJS with TS
- Jest
- Storybook
- Material UI
- NestJS
- SQL Database

# SETUP

1. Create `.env`file for `backend` and `frontend`

```
$ cp ./backend/.env.template ./backend/.env
```

```
$ cp ./frontend/.env ./frontend/.env.local
```

2. Run docker

```
$ docker compose up
```
